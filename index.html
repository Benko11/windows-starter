Meet Windows 11 Starter

Next generation of Windows

Our brand new version of Windows is built from the ground up to enable unique customer experiences even on the low-end devices. Experience the speed, security and reliability of Windows all in a single package.

Designed for the latest generation of Qualcomm Snapdragon A processors, Windows 11 Starter offers unmatched performance and features on comparably powerful hardware.

Ready when you are

Experience the speeds of Qualcomm Snapdragon X processors now on low-end devices. Instant wake and go to sleep, legendary battery life and compatibility with the applications that you need.

Windows 11 Starter delivers an experience unlike anything Windows has ever seen before.

Always up to date

Keep your Windows 11 Starter device secure by having Windows security updates regularly delivered. Powerful protection suite of Microsoft Defender automatically shields your device from remote malware attacks, so that you can always retain your peace of mind.

Stay on top of the newest features and get lifetime free Windows feature updates on the same timeline as Windows 11 Home or Pro.
